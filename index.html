1. เหตุผลที่อยากเรียน Full Stack Web Development

สาเหตุที่ฉันเลือกเรียน Full Stack Web Development เพราะอยากเข้าใจกระบวนการทำงานของเว็บไซต์ทั้งหมด ตั้งแต่การออกแบบหน้าตาเว็บไซต์ (Front-End) ไปจนถึงระบบหลังบ้านที่จัดการข้อมูล (Back-End) ฉันรู้สึกว่านี่คือทักษะที่สำคัญและสามารถนำไปต่อยอดทำโปรเจกต์ส่วนตัว หรือสร้างอาชีพในอนาคตได้

2. คาดหวังอะไรจากวิชานี้

ฉันหวังว่าจะได้เรียนรู้วิธีสร้างเว็บไซต์ตั้งแต่พื้นฐานไปจนถึงสามารถนำไปใช้งานจริง ได้เข้าใจการเชื่อมต่อระหว่าง Front-End และ Back-End รวมถึงการทำงานร่วมกับ GitHub และการทำงานแบบเป็นระบบเหมือนในสายงานจริง

3. อะไรคือทักษะที่คิดว่าจำเป็นเกี่ยวกับ Programming และแนวทางพัฒนา

ฉันคิดว่าทักษะที่สำคัญสำหรับการเขียนโปรแกรมคือ Logical Thinking การแก้ปัญหา (Problem Solving) และความอดทนในการลองผิดลองถูก แนวทางพัฒนาของฉันคือฝึกเขียนโค้ดอย่างสม่ำเสมอ ศึกษาจากเว็บไซต์ เช่น W3Schools, MDN หรือ YouTube และทำโปรเจกต์เล็ก ๆ เพื่อฝึกใช้ความรู้จริง
